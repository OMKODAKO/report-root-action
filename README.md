# 🌿 Take Care

**Take Care** არის civic-tech პლატფორმა, რომელიც ეხმარება მომხმარებლებს დააფიქსირონ გარემოსდაცვითი და ინფრასტრუქტურული პრობლემები ფოტოების, ლოკაციისა და რუკის საშუალებით.

პლატფორმის მიზანია ქალაქში არსებული პრობლემების ვიზუალური დაფიქსირება და საზოგადოების ჩართულობით მათი გაუმჯობესება.

---

## 🚀 პროექტის მიზანი

პროექტის მთავარი მიზანია:
- ნაგვის და დაბინძურებული ადგილების დაფიქსირება
- დაზიანებული ინფრასტრუქტურის მონიშვნა
- რესტავრაციის საჭიროების მქონე ობიექტების აღრიცხვა
- საზოგადოების აქტიური ჩართულობა გარემოს გაუმჯობესებაში

---

## ✨ ძირითადი ფუნქციები

### 📸 ფოტო გადაღება / ატვირთვა
- მომხმარებელს შეუძლია ფოტოს გადაღება კამერით
- ან უკვე არსებული ფოტოს ატვირთვა

### 📍 ლოკაციის დაფიქსირება
- GPS-ის ავტომატური გამოყენება
- რუკაზე პინის ხელით დამატება

### 🗺️ ინტერაქტიული რუკა
- ყველა რეპორტი ჩანს რუკაზე
- კატეგორიების მიხედვით ფერადი პინები
- დეტალური ინფორმაციის ნახვა

### 👤 მომხმარებლის სისტემა
- რეგისტრაცია / ავტორიზაცია
- პირადი პროფილი
- საკუთარი რეპორტების ისტორია

### 📰 Community Feed
- ყველა მომხმარებლის პოსტი ერთ სივრცეში
- რეალურ დროში განახლება
